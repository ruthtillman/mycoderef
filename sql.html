SQL Query Reference Sheet

A variety of SQL query examples, lots of basic stuff taken from basic tutorials as well as more personally-useful stuff.

Basic SQL Stuff

Base update/delete queries

UPDATE table_name
SET col1='val1', col2='val2'
WHERE key='keyvalue' [AND/OR] col2='val2';
DELETE FROM table_name
WHERE key='keyvalue' [AND/OR] col2='val2';

Insert Into

INSERT INTO table_name (col1, col2, col3)
VALUES (val1, val2, val3);

Columns don't have to specified in the first line, but they determine specific columns where things will be inserted. Depending on needs and specificity.

Unique results

SELECT DISTINCT col1, col2
FROM table_name
WHERE condition='fulfilled';

Order by

ORDER BY col1, col2 ASC|DESC;

Ascending by default. May order by all columns if decided.

SQL Joins

Fuuuuuuuucking SQL Joins. Anyway, in theory:

SELECT table_1.col_a, table_1.col_d, table_2.col_c
table_1 JOINTYPE table_2
    ON (table_1.col_a=table_2.col_f)
WHERE col_b='value'

Where JOINTYPE = the type of join— right, inner, etc.

The "database_name." can be affixed to the beginning of table names.

Inner Join

Returns a result row when both paired columns have matching values.

Outer Join

Returns a result row when either table has a column with the matching value.
